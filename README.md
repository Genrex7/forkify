## Installation

1. Clone the repo

    ```bash
    git clone https://github.com/Genrex7/forkify.git
    ```

2. Install NPM packages

    ```bash
    npm install
    ```

3. Obtain the APi key from [Forkify API](https://forkify-api.herokuapp.com/v2) and paste it in "src/js/config.js"

4. Run the project

    ```bash
    npm start
    ```
